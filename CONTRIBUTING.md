# Contributing to Awesome NSW Open Data

Thanks for helping make NSW government data more discoverable!

## How to contribute

### Adding a new dataset

1. Fork the repo and create a branch
2. Add the dataset to the appropriate section in `README.md`
3. Include: **name**, **URL**, **format**, and a brief **description**
4. Submit a PR with a short explanation of what the dataset contains and why it's useful

### What we're looking for

- NSW state government datasets, APIs, or data sources not already listed
- Data from NSW statutory authorities, commissions, and independent bodies
- Corrections to broken URLs (government links break constantly)
- Format or access updates (e.g., a dataset that was PDF-only is now available as CSV)
- New project ideas with specific data source references

### What we're NOT looking for

- Federal/Commonwealth datasets (unless specifically NSW-relevant, like ABS regional data)
- Commercial/paid data sources
- Datasets that require ethics approval or formal data sharing agreements to access
- Personal opinions on government policy

### Style guide

- Use Australian English
- Keep descriptions concise (one line where possible)
- Link to the most direct/stable URL (prefer permanent dataset pages over search results)
- For PDF-only data, note that in the format column
- Use `|` table formatting consistent with existing sections

### Reporting broken links

Open an issue with:
- The broken URL
- Which section it's in
- A replacement URL if you can find one

## Code of conduct

Be respectful and constructive. This is a data catalog, not a political forum.
